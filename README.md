# Smit Borasaniya — Portfolio

A minimal, sophisticated portfolio with a dark theme, cosmic aesthetics, and thoughtful transitions.

## Tech Stack

- **React 18** + **TypeScript**
- **Vite** — Build tool
- **Tailwind CSS** — Styling
- **Framer Motion** — Animations
- **Lucide React** — Icons

## Project Structure

```
src/
├── components/
│   ├── Hero.tsx           # Welcome & intro
│   ├── WelcomeGate.tsx    # Visitor name entry
│   ├── Navbar.tsx         # Navigation
│   ├── Interests.tsx
│   ├── Capabilities.tsx
│   ├── WorkEthic.tsx
│   ├── Work.tsx
│   ├── KeepInTouch.tsx
│   ├── Footer.tsx
│   ├── ScaleSection.tsx   # Scroll-driven section wrapper
│   ├── SectionProgress.tsx
│   ├── RotatingPhrase.tsx
│   └── CosmicBackground.tsx
├── context/
│   └── VisitorContext.tsx
├── App.tsx
├── main.tsx
└── index.css
```

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000/smit-portfolio/`

## Build

```bash
npm run build
```
